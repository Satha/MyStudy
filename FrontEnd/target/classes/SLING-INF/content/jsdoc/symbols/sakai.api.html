Convenience functions to aid Sakai 3 front-end development. This class is the basis of all Sakai 3 front-end development. This should be included on all pages, along with the sakai_api.js which is an extension of this class, providing higher level functions.

<static> {Date} sakai.api.l10ntoGMT(date)
Function that will take in a date object and will transform that date object into a GMT date object. This should always be done before we try to save a date back to a file or the database. The timezone we are currently in will be determined from our timezone set in the personal preferences page
Defined in: sakai.api.l10n.js.
Parameters:
{Date} date
JavaScript date object that we would like to transform in a GMT date object
Returns:
{Date} Date object, that will have transformed the given date and time into GMT date and time
