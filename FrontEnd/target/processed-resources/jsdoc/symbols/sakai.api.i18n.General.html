<static> {String} sakai.api.i18n.General.process(toprocess, localbundle, defaultbundle)
General process functions that will replace all the messages in a string with their corresponding translation. 
sakai.api.i18n.General.process(
    "<h1>__MSG__CHANGE_LAYOUT__</h1>",
    {"__MSG__CHANGE_LAYOUT__" : "verander layout"},
    {"__MSG__CHANGE_LAYOUT__" : "change layout"}
);
Parameters:
{String} toprocess
HTML string in which we want to replace messages. Messages have the following format: __MSG__KEY__
{Object} localbundle
JSON object where the keys are the keys we expect in the HTML and the values are the translated strings
{Object} defaultbundle
JSON object where the keys are the keys we expect in the HTML and the values are the translated strings in the default language
Returns:
{String} A processed string where all the messages are replaced with values from the language bundles
